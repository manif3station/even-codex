# DD-355

## Title

Implement and prove the hybrid glasses voice-query flow in `D2-Codex`.

## Method

`BDD`

## Acceptance

- the `D2-Codex` Even Hub app requests the documented microphone permission
  needed for a custom voice-input flow
- a glasses click can open the popup and start a supported speech-recognition
  session in the companion webview layer
- recognised speech updates the staged query and the glasses popup text in real
  time
- the existing swipe and click action flow can submit the recognised query
  without retyping it on the phone
- the app exposes a clear fallback message when browser speech recognition is
  unavailable
- Docker tests pass
- coverage reaches `100%`
- `cover_db` is cleaned
- `README.md`, `docs/`, `Changes`, and `tickets/` are updated
- release commit and push complete from the `even-codex` repo
