# 2026-05-31 Even Hub UX Upgrade

`D2-Codex` now ships a more usable Even Hub experience on both the phone-side plugin screen and the glasses page.

The phone-side plugin now provides:

- a connection summary instead of a bare origin form
- refresh, reset, and detail-cycle controls
- a readable setup checklist and pairing flow explanation
- endpoint summaries for the bridge health, bootstrap, and plugin URLs

The glasses-side UI now provides:

- separate header, detail, and footer containers instead of one dense block
- tap-to-refresh behavior from the header and footer
- tap-to-cycle detail panes for summary, network, and setup-step views
- explicit double-click exit guidance

This keeps the Hub app within the current Even layout constraints while making the bridge workflow much easier to understand in use.
