# 2026-05-31 Multi-Connector Session Control

`D2-Codex` now supports both multi-connector and multi-session control inside the Even app.

The phone-side plugin can now:

- save multiple local DD connector profiles
- switch the active connector without leaving the Even app
- keep a separate Codex session library for each saved connector
- activate or remove saved sessions for the active connector

The glasses page can now:

- refresh the active connector from the header
- cycle detail panes from the detail container
- switch to the next saved session for the active connector from the footer

Connector switching remains phone-only so the glasses interaction stays simple and low-risk.
