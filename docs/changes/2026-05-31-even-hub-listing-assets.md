# 2026-05-31 Even Hub Listing Assets

`D2-Codex` now ships the Even Hub listing material that was previously missing compared with the Weather-style Hub presentation.

This change adds:

- listing metadata under `even-hub/listing.json`
- longer Hub manifest fields in `app.json`
- a greyscale icon and background asset set
- a simulator-backed screenshot capture script and governed screenshot folder

The repo now carries both the runtime plugin UX and the listing/submission layer needed for Even Hub packaging and review preparation.
